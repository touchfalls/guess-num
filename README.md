# Number Guessing Game 🎲

Ссылка на проект ROADMAP: https://roadmap.sh/projects/number-guessing-game
**Угадай число** — простая консольная игра на Python.  

Компьютер загадывает случайное число от 1 до 100, а ваша задача — угадать его.  
Выбирайте уровень сложности, который определяет количество попыток.

---

## 🎮 Особенности игры

- Три уровня сложности:
  - **Easy** — 10 попыток
  - **Medium** — 5 попыток
  - **Hard** — 3 попытки
- Игра сообщает, если ваша догадка **слишком высокая** или **слишком низкая**
- Конец игры наступает, когда число угадано или попытки закончились
- Лёгкий старт для новичков в Python

---

## 🚀 Как запустить

1. Клонируйте репозиторий:

```sh
git clone https://github.com/touchfalls/guess-num.git
```

2. Перейдите в директорию проекта:
```
cd guess-num
```

3. Запустите игру с помощью Python:
```
python number_guessing_game.py
```
