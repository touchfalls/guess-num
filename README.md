# Number Guessing Game

**Угадай число** — простая консольная игра на Python.  

Компьютер загадывает случайное число от 1 до 100, а игроку нужно его угадать.  
Игрок выбирает уровень сложности, который определяет количество попыток.

---

## 🎮 Особенности игры

- Три уровня сложности:
  - **Easy** — 10 попыток
  - **Medium** — 5 попыток
  - **Hard** — 3 попытки
- Игра сообщает, если ваша догадка слишком высокая или слишком низкая
- Конец игры наступает, когда игрок угадывает число или заканчиваются попытки
